    Section 1.5 Bimodules

    Suppose that \(A\) is a right \(R\)-algebra, and that \(N\) is a right \(A\)-module. Then, \(N\) is also a right \(R\)-module. Similarly, if \(B\) is a left \(R\)-algebra, and \(N'\) is a left \(B\)-module, then \(N'\) is also a left \(R\)-module.

    Definition 1.5.1. (A,B) bimodule.

    Let \(R\) be a commutative ring with unity. Let \(A\) and \(B\) be right \(R\)-algebras. An \((A,B)\)-bimodule is an abelian group \(M\) which is both a left \(A\)-module and a right \(B\)-module such that for all \(a\in A\text{,}\) \(b\in B\text{,}\) \(r\in R\) and \(m\in M\) we have the following compatibility conditions.
    \begin{equation*} (am)b = a(mb)\quad\text{and}\quad rm = mr \end{equation*}

    Example 1.5.4. Representations and left modules.

    Let \(R\) be a commutative ring with unity, and let \(A\) be a right \(R\)-algebra. If \(M\) is a left \(A\)-module, then \(M\) is an \((A,R)\)-bimodule.
    ðŸ”—
    We can define representation of \(L\colon A\to\End{R}{M}\) by \(a\mapsto L_a \,(m\mapsto a\cdot m)\text{.}\) The map \(L\) is an \(R\)-algebra homomorphism.
    ðŸ”—
    Conversely, if \(\varphi\colon A\to\End{R}{M}\) is an \(R\)-algebra homomorphism, then \(M\) is a left \(A\)-module via \(a\cdot m = \varphi(a)(m)\) for all \(a\in A\) and \(m\in M\text{.}\)
